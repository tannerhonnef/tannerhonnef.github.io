Undergraduate Honors Thesis

Project Description:

This project served as my undergraduate honor's thesis and earned highest honors advised by Professor Robert Gilmore Pontius Jr. This work was presented at AAG in April 2024, AAAS in February 2025, and IALE in April 2025. The work establishes a set of best practices for using TOC curves and evaluates 10 papers on thier use of those best practices when generating TOC curves. It also dicusses improvements for TOC when used with machine learning algorithms.
